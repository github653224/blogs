> frame 框架切换技巧

>> 切换到上一层框架

```python
driver.switchTo().parentFrame()
```

>> 切回上一层框架

```python
driver.switchTo().defaultContent()
```

> 刷新页面

```python
driver.navigate().refresh();
```

> 关闭当前页面

```python
driver.close();
```
