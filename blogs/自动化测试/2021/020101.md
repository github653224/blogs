> frame 框架切换技巧

>> 切换到上一层框架

```python
from selenium import webdriver
driver = webdriver.Chrome()
driver.switchTo().parentFrame()
```

>> 切回上一层框架

```python
from selenium import webdriver
driver = webdriver.Chrome()
driver.switchTo().defaultContent()
```

> 刷新页面

```python
from selenium import webdriver
driver = webdriver.Chrome()
driver.navigate().refresh();
```

> 关闭当前页面

```python
from selenium import webdriver
driver = webdriver.Chrome()
driver.close();
```
